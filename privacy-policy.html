Privacy Policy

Last updated: March 2026

This Privacy Policy describes how Stylocorner ("we", "us", or "our") handles information in connection with our Pinterest affiliate marketing tool for Indian ethnic fashion.

1. About This Application

Stylocorner is a personal affiliate marketing tool that creates and posts Pinterest pins for Indian ethnic fashion products available on Amazon India. The application helps promote traditional Indian clothing by creating curated Pinterest content with affiliate links.

2. Information We Collect

We do not collect any personal information from users. This application is operated solely by its owner and does not have a user registration system, login system, or any mechanism to collect personal data from third parties.

The only data processed by this application includes:

  • Publicly available product information from Amazon India (titles, images, prices, ratings)
  • Pinterest account information of the app owner for the purpose of posting pins

3. How We Use Pinterest API

This application uses the Pinterest API solely to:

  • Create pins on the app owner's Pinterest boards
  • Post product images with affiliate links
  • Add AI-generated titles and descriptions to pins

We do not access, store, or share any Pinterest user data beyond what is necessary to post pins on the owner's account.

4. Cookies

This application does not use cookies or any tracking technologies.

5. Third Party Services

This application interacts with the following third party services:

  • Pinterest API — for posting pins to Pinterest boards
  • Amazon India — for retrieving publicly available product information
  • Google Gemini API — for generating pin titles and descriptions

Each of these services has their own privacy policies which govern their data handling practices.

6. Data Storage

This application does not store any user data in a database. All product information is fetched in real time and used temporarily during a single session. No data is retained after the session ends.

7. Data Sharing

We do not sell, trade, or share any data with third parties. The application does not have the capability to collect or transmit personal information of any kind.

8. Children's Privacy

This application is not directed at children under the age of 13 and does not knowingly collect any information from children.

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated revision date.
